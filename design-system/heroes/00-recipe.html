Hero Images

The recipe

Every project hero is a 1600×900 SVG built from one shared template, so the whole set reads as a single system. Only the central motif changes per project.

  1. Background — vertical navy gradient (#0c1122 → #070a13).
  2. Dual bloom — warm amber radial top-right, faint cool slate radial bottom-left.
  3. Grid — 1px amber lines at 7% opacity, masked to fade toward the edges.
  4. Motif — the per-project diagram; amber-stroked nodes carry a soft glow.
  5. Vignette — radial edge-darkening to focus the center.
  6. Grain — fractal-noise overlay at ~5%.
  7. Accent bar + type — end-faded amber rule, eyebrow (amber caps), serif title.
